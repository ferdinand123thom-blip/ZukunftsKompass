# Zukunftskompass

Erster statischer Website-Start für den Zukunftskompass: eine Wissensplattform für junge Menschen zu Themen, die nach der Schule wichtig werden.

## Struktur

- `index.html` - Startseite
- `assets/css/styles.css` - Design und responsives Layout
- `assets/js/main.js` - kleine Interaktionen
- `assets/images/` - lokale Bilddateien
- `data/topics.json` - vorbereitete Themen-Daten für spätere Erweiterungen

## Lokal starten

Die Seite ist aktuell statisch. Du kannst `index.html` direkt im Browser öffnen.

Für späteres Hosting:

1. Projekt bei GitHub hochladen.
2. Repository mit Vercel verbinden.
3. Als Framework `Other` bzw. statische Seite verwenden, solange noch kein Build-System genutzt wird.
